# Implement Feature (Frontend)

1. **Input**:
   - Ask the user for the feature description if not provided.
2. **Implement**:
   - Run the `implement-feature` skill with the description.
   - This skill will:
     - Create components/services following `angular-standards`.
     - Implement the logic.
     - Write unit tests (`write-tests`).
     - Update documentation.
3. **Verify**:
   - Ensure tests pass and the code adheres to `project-conventions`.
