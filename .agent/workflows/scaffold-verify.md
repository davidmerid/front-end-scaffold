# Verify Scaffold

1. **Verify**:
   - Run the `scaffold-verify` skill.
   - It will scan the codebase for Hexagonal Architecture violations (e.g., framework code in domain layer), missing files, and other Gold Standard drifts.
2. **Report**:
   - The skill will produce a compliance report.
   - Review the findings and fix any critical violations immediately.
